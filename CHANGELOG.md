# Changelog

## 0.2.0 — Single-Label MVP Implementation

- Added FastAPI backend with health, OCR, and verification endpoints.
- Added Tesseract/pytesseract OCR pipeline with Pillow preprocessing.
- Added deterministic matching engine for Brand Name, Class/Type, Alcohol Content, Net Contents, and Government Warning.
- Added mandatory government warning validator.
- Added PASS / FAIL / REVIEW field-level result model.
- Added React/Vite/TypeScript frontend with upload, application form, dashboard, and OCR text display.
- Added sample label generator and sample application JSON.
- Added Docker deployment support with Tesseract installed.
- Verified backend tests pass and frontend builds.
