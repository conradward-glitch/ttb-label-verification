# Project Overview

## Project

Treasury / TTB AI-Powered Alcohol Label Verification App

## Purpose

This take-home project demonstrates a practical AI-assisted workflow for alcohol label review. The app helps TTB compliance agents compare uploaded label artwork against application data and identify mismatches in routine fields.

## Core Workflow

1. User uploads a label image
2. OCR extracts text from the label
3. User enters expected application data
4. App verifies label text against expected values
5. App displays PASS / FAIL / REVIEW results

## Target Users

- Compliance agents reviewing label applications
- Supervisors evaluating routine verification workload
- IT stakeholders assessing feasibility of standalone AI-assisted review tools

## MVP Features

- Image upload
- OCR extraction
- Application data entry
- Field-by-field matching
- Government warning verification
- Results dashboard
- PASS / FAIL / REVIEW outcome

## What This Is Not

- Not a COLA replacement
- Not a production federal system
- Not a compliance decision-maker without human review
- Not a microservices or enterprise-cloud architecture
- Not a permanent document storage system

## Success Criteria

The prototype succeeds if a reviewer can quickly understand it, run it, upload or select a sample label, and see useful verification results with clear explanations.
